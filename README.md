# Mini Calculator 🧮

This is a basic Python calculator that supports:

- Addition
- Subtraction
- Multiplication
- Division (with zero-check)
- Looping for multiple calculations

## Features
✅ Uses input(), int(), and if-else  
✅ Protects against division by zero  
✅ Built with beginner-friendly logic
